How it works

From install to your first color in under 30 seconds

No setup. No tutorial. No learning curve. Just install, press, click.

1

Install from the Chrome Web Store

One click installs Color Chooser in seconds. No account, no signup. The extension adds a tiny toolbar icon and registers a global keyboard shortcut.

2

Press Alt + Q on any web page

The eyedropper activates instantly. A magnifier loupe follows your cursor, showing the exact pixel you're hovering over and its hex code in real time — before you even click.

3

Click — done

The hex code is in your clipboard before your mouse button comes back up, and a 'Color Copied' notification confirms it. Paste it into Figma, your CSS, Slack, an email — anywhere. Your last 20 picks stay saved in the side panel.
